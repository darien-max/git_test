 LISTADO DE CODIGOS

-git init //sirve para empezar un archivo para git
-git log //sirve para verificar todos los avances y commits
-git status //muestra los archivos que han sido modificados
-git add //sirve para verificar los archivos
-git add . //sirve para guardar todos los archivos y canbios en git
-git remote add origin https://github.com/OWNER/REPOSITORY.git //sirve para relacionar la cuenta de git 
-git push -u origen master //sirvesirve para conectar la cuenta de git 
-git commit -m"primer commit" //se usa para crear el primer o la cantidad de commits q uno quiera
-clear //sirve para limpiar la pisarra de git
-git cofig credential.user.name"Darien" //se usa para cambiar o actualizar el usuario de git
-git cofig credential.useremail dpalaciosg@inemkennedy.edu.co //se usa para cambiar o actualizar el email de git
-git cofig --global user.name"your name" //sirve para crear tu usuario de git
-git cofig --global user.email dpalaciosg@inemkennedy.edu.co //sirve para crear el email de git
-git cofig --global user.name //se usa para verificar el usuario de git
-git cofig --global user.email //se usa para verificar el email de git
-git push //seutiliza para subir canvios al repositorio o crear cambios en el repositorio o rama
